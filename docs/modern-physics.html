⚡ Special Relativity

Postulates

  1. Laws of physics same in all inertial frames
  2. Speed of light c constant in all inertial frames

Consequences: Time dilation, length contraction, relativity of simultaneity

Lorentz Transformations

x' = γ(x - vt)
t' = γ(t - vx/c²)
γ = 1/√(1 - v²/c²)

Time dilation: Δt = γΔt₀
Length contraction: L = L₀/γ

Relativistic Dynamics

E = γmc² = √((pc)² + (mc²)²)
p = γmv

Rest energy: E₀ = mc²
Kinetic energy: K = (γ-1)mc²

☢️ Early Quantum Theory

Blackbody Radiation

Planck's Law: Energy quantized in packets E = hf

h = 6.626×10⁻³⁴ J·s (Planck's constant)

Resolved ultraviolet catastrophe

Photoelectric Effect

Einstein (1905): Light as photons

K_max = hf - φ

φ = work function
Explained why low-frequency light doesn't eject electrons regardless of intensity

Compton Scattering

λ' - λ = (h/m_e c)(1 - cos θ)

Confirmed photon momentum p = h/λ

Demonstrated particle nature of light

de Broglie Waves

λ = h/p

All matter exhibits wave properties

Confirmed by electron diffraction experiments

⚛️ Atomic Physics

Bohr Model

Quantized angular momentum: L = nℏ, n = 1,2,3,...

E_n = -13.6 eV/n²
r_n = n²a₀ (a₀ = 0.529 Å)

Photon emission: hf = E_i - E_f

Hydrogen Spectrum

Lyman series: n_f = 1 (UV)
Balmer series: n_f = 2 (visible)
Paschen series: n_f = 3 (IR)

Rydberg formula: 1/λ = R_H(1/n_f² - 1/n_i²)

☢️ Nuclear Physics

Nuclear Structure

Nucleons: Z protons + N neutrons = A
Notation: ᴬ_Z X
Binding Energy: BE = [Zm_p + Nm_n - M]c²

Radioactive Decay

N(t) = N₀e^(-λt)

Half-life: t₁/₂ = ln(2)/λ
Alpha decay: ᴬ_Z X → ᴬ⁻⁴_Z₋₂ Y + ⁴₂He
Beta decay: n → p + e⁻ + ν̄_e
